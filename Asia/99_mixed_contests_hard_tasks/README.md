# Mixed Contests and Hard Tasks

Folder nay khong hoc theo tag. Muc tieu la luyen pipeline that:

1. Doc statement lau va tach constraints.
2. Tim model.
3. Chung minh invariant / necessary-sufficient condition.
4. Chon ky thuat.
5. Code dai nhung co checkpoints.
6. Debug bang sample, corner, stress/brute neu co the.

## How to use

- Contest set: lam dung gio, khong mo tag/editorial.
- Hard task: set timer 2-4 gio. Neu sau 40 phut chua co model, viet lai cac huong da thu.
- Upsolve note dat trong `../upsolve/`.

- [ ] **Wiring**  
  Submit: [OJ.uz IOI17_wiring](https://oj.uz/problem/view/IOI17_wiring)  
  Mix: sorted DP, grouping, optimization.

- [ ] **Holiday**  
  Submit: [OJ.uz IOI14_holiday](https://oj.uz/problem/view/IOI14_holiday)  
  Mix: DP, data structure, divide structure.

- [ ] **Game**

- [ ] **Seats**

- [ ] **Teams**

- [ ] **Shortcut**

- [ ] **Commuter Pass**

- [ ] **One-Way Streets**

- [ ] **Fancy Fence**

- [ ] **Palindromic Partitions**

- [ ] **Cave**
