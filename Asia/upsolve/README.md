# APAC Upsolve Queue

Giữ cùng format với `regional/upsolve/`, thêm hai trường:

```text
APAC value: likely scoring / stretch / one-team problem
Decision: implement now / learn primitive first / archive insight only
```

Không bắt buộc code ngay các bài cực hạn. Ưu tiên bài có số đội giải trung bình và kỹ thuật chuyển giao được sang Regional/APAC khác.
