                                       Problem A
                                      Cloud System
                                  Time Limit: 3 seconds
                                 Mem limit: 512 Megabytes
VNU-HCM University of Science has just built a new cloud computing system to run experiments
of research projects related to deep-learning.
The system consists of 𝑁 servers (numbered from 1 to 𝑁). Each pair of servers is connected by at
most 1 cable (possibly 0).

nanosecond.
Let us define 𝐹𝑢,𝑣 as the data transfer rate between server 𝑢 and server 𝑣. To transfer data from
server 𝑢 to server 𝑣, data can be split into multi parts and transferred via multiple routes.

                             1 - 4 (30 Mb per nanosecond)
                             1 - 3 - 4 (20 Mb per nanosecond)

                      so the transfer rate from 1 to 4 is 60 Mb per nanosecond or 𝐹1,4 = 60.


You are given the configuration of the network and the transmission capacity of cables, your task

Note: The sum of 𝑁 in the input does not exceeded 1000.

Output
