# Virtual Logs

Tạo một file cho mỗi contest, ví dụ `2026-06-20_southern-2023.md`.

```text
Contest:
Date:
Team:
Final score:

Timeline:
- 00:00-00:30:
- First AC:
- Last AC:

Problem table:
| ID | Reader | Owner | Status | Time | WA | Root cause |

What worked:
What failed:
Wrong solve order:
Communication issue:
One-machine bottleneck:
Actions before next virtual:
```

Các chỉ số cần theo dõi qua từng tuần:

- Số AC sau 60, 120, 180 và 300 phút.
- Số WA trên bài cuối cùng AC.
- Thời gian từ có proof tới submit.
- Số bài bị hai người cùng code hoặc không ai owner.
