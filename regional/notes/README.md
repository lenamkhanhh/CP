# Notes

```text
Topic:
Trigger: dấu hiệu nào trong statement gợi ra kỹ thuật?
Invariant/state:
Transition/model:
Proof:
Complexity:
Implementation traps:
Related problems:
```

Không chép lại editorial. Một note tốt phải đủ ngắn để đọc lại trong 5 phút trước contest.
